# Java Exercises Repository

Repositório dedicado ao armazenamento e organização de exercícios em **Java**, com foco no desenvolvimento de lógica de programação, estruturação de código e boas práticas.

## Sobre o projeto

Este repositório contém uma coleção de exercícios desenvolvidos ao longo dos estudos em Java. O objetivo é reforçar conceitos fundamentais e evoluir gradualmente para tópicos mais avançados da linguagem.

Os exercícios abordam desde lógica básica até conceitos mais estruturados da programação orientada a objetos.

## Tecnologias utilizadas

* Java
* JDK 8+
* IDEs (IntelliJ IDEA / VS Code / Eclipse)

## Conteúdos abordados

* Variáveis e tipos de dados
* Estruturas condicionais (if/else, switch)
* Laços de repetição (for, while, do-while)
* Métodos e funções
* Programação Orientada a Objetos (POO)
* Arrays e Collections
* Tratamento de exceções


## Objetivo

* Praticar lógica de programação
* Consolidar conhecimentos em Java
* Evoluir na resolução de problemas
* Criar base sólida para projetos maiores

Este repositório será atualizado continuamente com novos exercícios e desafios conforme o avanço nos estudos.
